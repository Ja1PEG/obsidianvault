## Hanoi Tower Solution:

```c++
void towers(int n, char apeg, char bpeg,char tpeg)

{
    if (n == 0) {

        return;

    }

    towers(n - 1, apeg, tpeg, bpeg);

    cout << "Move disk " << n << " from peg " << apeg<< " to peg " << bpeg << endl;

    towers(n - 1, tpeg, bpeg, apeg);
}
```
